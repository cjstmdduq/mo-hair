## **[웹사이트 구축 초도 기획안]**

### 브랜드·포지셔닝

- **상호:** 모헤어 의원
- **영문 표기:** MO-HAIR 의원
- **한 줄 소개:** 남성형 탈모·모발이식 전문

### 기본 사양

- **구조:** 반응형 원페이지(One-page)
- **폰트:** Pretendard
- **컬러:** Primary `#3C5E64` / 화이트 / 그레이(Neutral)
- **모서리:** 라운드 **강한 편** (카드·버튼·미디어 블록 통일)

### 섹션 구성

- **Hero**
    - 카피: 비절개 2000모 399만 / 절개 2000모 299만
    - 서브: 남성형 탈모·모발이식 전문(브랜드 한 줄과 톤 맞춤)
    - 인터랙션: 배경 이미지 스왑(Swap)
- **Cases (시술 케이스)**
    - 메인 3~4건 노출
    - 게시판형 리스트 또는 고정 그리드
- **Director (의료진 소개)**
    - 프로필 사진 + 약력/이력
- **Procedures (시술 정보)**
    - 비절개 / 비절개 무삭발 / 절개
    - 시술별 특징 + 차이점 비교 데이터
- **Contents (콘텐츠)**
    - 유튜브 영상 임베드 + 블로그 주요 포스팅 리스트
- **Location (오시는 길)**
    - 네이버 지도 API + 네이버 플레이스 링크
- **Footer**
    - 병원 정보 / 사업자 정보 / 개인정보처리방침 등 필수 고지

### 스타일 가이드(초안)

- **타이포:** H 28~36 Bold / B 15~16 Regular / LH 1.5~1.7
- **컬러**
    - **Primary:** `#3C5E64` (버튼·헤더·강조 라인·링크 등)
    - **Neutral:** 화이트·라이트그레이(배경·구분)
    - **Accent:** CTA 보조 1색(Primary와 대비·가독 확보, 과다 사용 금지)
- **레이아웃:** 최대폭 1100~1200 / 섹션간 72~120 / 모바일 패딩 16~20
- **컴포넌트**
    - 버튼 2종(Primary/Outline), **라운드 강한 편** — 소형 14~18 / 중형·카드·이미지 20~28 / 풀블리드 섹션 내부 블록은 24~32 권장
    - 그림자 최소
    - CTA는 1개만(상단 또는 하단)
- **이미지/문구:** 톤 통일·과한 필터 금지 / 짧고 단정 / 가격·혜택 문구는 법무 체크 전제
